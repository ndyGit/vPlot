Defined in: ../core/utils.vlib.js:3
Module: VLib

Base class of the framework.
On the one hand vLib is an interface to the "outside world". On the other it provides information to modules.
VLib registers it self as pseudo module and acts as a service provider.
It holds an instance of VMediator.

Array

Defined in ../core/utils.vlib.js:111

Scales data linear with respect to the given borders.
Scale function is (data(x,y,z) / range(x,y,z)) * targetRange(x,y,z).
range_ is ZERO to max(border.max,abs(border.min))

Parameters:

  • data Array 
    Array of data objects {x:VALUE,y:VALUE,z:VALUE}.
